## 项目说明

在前端领域，有很多很经典的书籍，这个仓库的目的就是为了记录在学习前端这些经典书籍的过程中，收获的一些很重要的概念或者知识点，将他们收集起来，方便随时查阅，日积月累，相信我们每个人都可以成为前端大牛
